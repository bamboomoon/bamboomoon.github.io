Comparable 作用

实现这个 protocol，让你自定义的对象可以像 Int, String 一样可以比较大小。当然，也可以排序，也可以放在元组中，比较元组的大小

实现起来还是比较简单的，直接放代码

注意

  • < 和 == 是必须实现的，Comparable 是继承于 Equatable的
  • 最后实现出来的比较必须满足3个条件： 非自反性、不对称性、传递性
  • 接上一条：你实现了 == < <= 就相当于实现了 != < >=
  • 如果 <= 或者 >= 没有实现，也是可以使用的。使用的结果： 它会调用你实现的 < 函数，将得到的结果取反，然后返回
